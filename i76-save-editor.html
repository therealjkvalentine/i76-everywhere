Where do the save files live? What do the fields mean?

Easiest: run i76-save-editor.command (or python3 i76-save-editor-server.py [--dir DIR]) from the repo — it opens this page with your saves already on the order pad and lets Save Bookmark write straight back to the game folder (one-time .pre-edit backups). Without it, drag the files in from:

  • Mac (this repo's wrapper): ~/Applications/Sikarugir/Interstate 76 - Software (DxWnd).app/Contents/SharedSupport/prefix/drive_c/GOG Games/Interstate 76/
  • Windows (GOG): next to i76.exe (e.g. C:\GOG Games\Interstate 76\)
  • Steam Deck (Heroic): the game folder inside the install path

In drag-and-drop mode, downloads replace the originals by hand. "Save as" a new slot also downloads an updated savegame.dir — copy both in, or the LOAD screen won't list the new slot.

Format notes (reverse-engineered, see the repo docs): the save holds the car (equipped parts, armor in tenths, exactly as the DEFENSE panel shows), the van inventory (spare parts — the game marks these (V)), and a trailing section we read as the repair order (R). Ammo is never stored — bookmarks happen at the shack, weapons refill from their spec. Each part carries a location flag — (C) in/on the car, (R) in repair, (V) in the van — and condition points vs durability drive the game's highlight colors (green scuffed → red wrecked; full points = no highlight). Don't mark everything (C): the game piles it into the car and unmounts hardpoints — "Stow spares in van" fixes that. The scene number in savegame.dir is the last COMPLETED scene; loading plays the next one (the game writes the newest entry's scene truncated, so check the Scene box before saving — a 0 loads at Scene 1 with the scripted starter loadout). Weapon range and DPS come from Local Ditch's weapons page (community measurements by Greg Schwartz / Zaphod-AVA, in "points of armor" — the same tenths the DEFENSE panel uses); where no measurement exists we show "~N dps spec" computed from the game's own weapon files (damage × salvo × shots/sec ÷ 10, all shots hitting — real DPS runs lower with misses). Flamethrowers are rated the community way: time to destroy a bus.

Glovebox: the classic cheat codes

All work in I'76 and Nitro Riders (source: the I'76 fandom wiki). Hold Ctrl+Shift and type, during play:

  • getdown — level skip (Groove says "get down"; everyone attacks you; die and you restart on the next scene)
  • thirdnostril — bigger radar  ·  wiggleburger — blurry vision
  • with the map up: frflat / flflat / brflat / blflat — deflate that tire
  • Ctrl+Alt+X — self-destruct  ·  F12 on landing — survive a huge fall

Melee-variant names: save a Rattler variant as retpocileh (helicopter), knat (tank); a Phaedra Rattler as hotair (balloon) or thetruthishere (UFO — you saw it crash outside Roswell, after all).
